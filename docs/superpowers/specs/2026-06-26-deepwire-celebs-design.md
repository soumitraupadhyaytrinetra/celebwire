Third sibling of DeepWire (AI/deep-tech) and FinWire (finance/markets), focused on **global celebrity and entertainment news**. **Hollywood (5):** Variety, Hollywood Reporter, Deadline, Entertainment Weekly, Just Jared

Risk to surface: many of these sites have moved their RSS endpoints in the past year (especially Deadline, EW, Billboard, Pitchfork). The scaffold step probes every URL and reports which survived; dead ones are swapped or removed.

## 5. Classifier Rewrite

`src/lib/ai/classify.ts` is rewritten with celebrity vocabulary:

- `CATEGORY_KEYWORDS`: each category gets ~30–60 distinctive keywords (e.g. `bollywood` includes "bollywood", "shah rukh khan", "salman khan", "deepika padukone", "amitabh bachchan", "hindi cinema", "indian film industry", etc.; `kpop` includes "k-pop", "bts", "blackpink", "twice", "stray kids", "korean idol", "comeback", etc.).
- `SOURCE_CATEGORY_HINTS`: one entry per source — primary category for that feed. If `getPrimaryCategory` finds no keyword hits, the hint picks the category.
- `categoryMapping`: identity mapping for the 5 new IDs.

## 9. Risks & Mitigations

| Risk | Mitigation |
|---|---|
| Several candidate feeds 404 / don't return RSS | Probe all URLs during scaffold, swap or drop failures. Worst case: 4–5 categories still have feeds. |
| TMZ, Just Jared rate-limit crawlers | Crawler already has `1500ms` delay between articles; may need to bump to `3000ms` if 429s appear. |
| Bollywood Hungama / Filmfare use non-standard XML | `xml2js` strict mode already off; if issues, switch to a tolerant parser. |
| Build error from finance-specific pages | Delete all 8 finance category page files explicitly before build. |
